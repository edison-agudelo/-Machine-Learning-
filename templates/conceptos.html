Conceptos básicos de Regresión Lineal

La regresión lineal es un método estadístico utilizado para modelar la relación entre una variable dependiente y una o más variables independientes mediante el ajuste de una línea recta que minimiza las diferencias entre los valores observados y los predichos. Esta técnica se aplica para predecir, analizar y comprender cómo las variables independientes influyen sobre la variable dependiente (Wikipedia, 2025).

Tipos de regresión lineal

  • Regresión lineal simple: Involucra una sola variable independiente y una variable dependiente. La relación se representa mediante una ecuación lineal:
    y = β₀ + β₁x + ε, donde y es la variable dependiente, x la independiente, β₀ el intercepto, β₁ la pendiente y ε el término de error.
  • Regresión lineal múltiple: Implica dos o más variables independientes para predecir la variable dependiente. La ecuación general es:
    y = β₀ + β₁x₁ + β₂x₂ + ... + βₙxₙ + ε, donde cada βᵢ representa la influencia de la variable independiente xᵢ sobre y.

Supuestos principales

  • Linealidad: La relación entre las variables independientes y dependientes es aproximadamente lineal.
  • Independencia: Los errores del modelo (residuos) son independientes entre sí.
  • Homoscedasticidad: La varianza de los errores es constante.
  • Normalidad: Los errores se distribuyen normalmente.

Aplicaciones comunes

  • Predicción de ventas en función de publicidad o tiempo.
  • Estimación del rendimiento académico según horas de estudio.
  • Análisis de riesgos financieros y modelos de inversión.
  • Estudios de satisfacción de clientes según factores de servicio.
